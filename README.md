# Conway's Game of Life

An interactive browser-based implementation of Conway's Game of Life. Draw cells, run or step the simulation, load classic patterns, change the speed and grid size, and share board states using a URL.

## Play online

https://fhegouaburu.github.io/Conways-Game-Of-Life/

## Publish with GitHub Pages

1. Create a **public** GitHub repository named `conways-game-of-life`.
2. Upload `index.html`, `.nojekyll`, `LICENSE`, and this `README.md` to the top level of the repository.
3. Open **Settings → Pages**.
4. Set **Source** to **Deploy from a branch**.
5. Select **main** and **/(root)**, then click **Save**.
6. Open the play link above after GitHub finishes deploying.

## Run locally

Double-click `index.html`. The entire app is contained in that one file.

## Controls

- Click or drag to draw cells.
- Right-click or Shift-drag to erase.
- Press **Play**, **Step**, **Random**, or **Clear**.
- Press the space bar to play or pause.

## Technology

Vanilla HTML, CSS, and JavaScript. No framework, build command, backend, or installation is required.
